Adding an HTML Editor for a Field

In order to add an HTML Editor for an Editor Field in the Admin Area, you must do the following:

  • Add the Attribute "[UIHint("tinymce_jquery_full"), AllowHtml]" in the Model definition right above the column name.
  • Add a reference to "System.Web.Mvc" and "System.ComponentModel.DataAnnotations" libraries.

The TinyMCE Editor was installed using the Package Manager by running the following command:

 

        PM> Install-Package TinyMCE.MVC.JQuery
